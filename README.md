# 🏭 MSSO Industries Website

A professional, responsive multi-page website developed for **MSSO Industries** to showcase their industrial services, clients, and company profile.



## 🚀 Features

* 🌐 Multi-page layout (Home, About, Services, Clients, Gallery, Contact)
* 📱 Fully responsive design
* ⚡ Optimized images (`.webp`, `.avif`) for faster loading
* 🏢 Industrial-focused UI with clean layout
* 📷 Gallery section for showcasing work
* 📞 Contact page for customer inquiries



## 📂 Project Structure

```
MSSO_industries/
│── index.html
│── about.html
│── services.html
│── clients.html
│── gallery.html
│── contact.html
│── images/
```



## 🛠️ Tech Stack

* HTML5
